
# SkinGuessr

SkinGuessr is a fun and engaging web application designed for the Counter-Strike (CS) community of skin enthusiasts.
## Roadmap

- Score System

- Increasing the pool of available skins

- Global Leaderboard


## Tech Stack

**Client:** Next.js, React, TailwindCSS

**Server:** Next.js
